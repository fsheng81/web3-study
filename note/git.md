

github 推送

```bash
git clone xxxxx
git status
git add .
git commit -m "xx" 
git commit --amend
git push
```

本地git仓推送到GitHub

```bash
# github 建立没有readme.md的仓库
# 本地建立Git仓
# 建立本地仓与远程仓的联系
git push -u origin main
```

github 切换branch

```bash
git branch 
git checkout -b dev2 # 建立branch
git checkout main # 切换branch
# 本地branch与远程建立联系
git push --set-upstream origin dev2
```
